# Plantify - Frontend Internship Assessment

A modern responsive plant ecommerce landing page built using Next.js, TypeScript, and Tailwind CSS.

## Live Demo

Add your Vercel deployed link here.

## Features

- Responsive modern UI
- Mobile hamburger navigation
- Reusable React components
- Dynamic rendering using data arrays
- Responsive grid layouts
- Interactive hover animations
- Optimized images using Next.js Image component
- Production-ready architecture

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS
- React Icons

## Folder Structure

```bash
app/
components/
data/
public/images/
```

## Installation

```bash
npm install
npm run dev
```

## Build For Production

```bash
npm run build
```
